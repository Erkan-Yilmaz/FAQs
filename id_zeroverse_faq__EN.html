What is id@Zeroverse ?

 To use all features in most ZeroNet sites/apps it's required that you log in:
  • e.g. you can post a msg on Zerotalk only after logging in with your ID,
    but you can read - without logging - in everything.
id@ZeroVerse is a trusted authorization provider,
and allows you to interact with ZeroNet sites
without contacting the owner.

What does the word ZeroVerse mean?
  • It stands for: the zero universe
  • You can find here past projects from the ZeroVerse initiative


Register your ID via: mail, BitMessage or IRC

 You can register here an ID:
  • inside ZeroNet: see the mail apps or
  • outside:
    • BitMessage or
    • IRC
If you really want to stay anonym,
you'd use BitMessage.

inside ZeroNet: mail apps

Before we talk about the register process via mail:
  • Do NOT use a proxy/gateway!
  • Why?
    • The proxy operator can access everything (your ID, data, ...).
    • When the proxy reboots: you've lost your ID


Registering your ID via the Mail apps is easy, because you don't leave the ZeroNet network.
Keep in mind:
  • but then I'd also know which ZeroID the new Zeroverse ID is tied to :-(
I'm not saving the mails anyways - or any kind of data - which could track back to you.
But, it's always better to be careful!

IRC

IMPORTANT:
  • Do NOT post any sensitive data into the public irc channel,
    others can see this and are then able to register first with your data (e.g. via mail or BitMessage).
  • If you do not get at the IRC server a registered nick, people can see your IP.


step 1:

This link will connect you to the freenode network (into the channel #zeroverse) via browser.
    If you've an irc client, you can also switch to the channel via this command:
    • /join #zeroverse
step 2:

Make sure that you are really talking with me:
verify that you are talking with user Erkan_Yilmaz by executing this command:
  • /whois Erkan_Yilmaz
The result will be similar to:
  • [Erkan_Yilmaz] (~Erkan_Yil@wikimedia/Erkan-Yilmaz): Erkan Yilmaz
    ...
    [Erkan_Yilmaz] is using a secure connection
    ...
    [Erkan_Yilmaz] is logged in as Erkan_Yilmaz
When I'm available, I will open a private chat with you:
  • /msg your_username
step 3:

When I'm not available:
  • just say Hello to me in the public channel, or
  • send me a private message via: /msg Erkan_Yilmaz Hello, I want to register an ID.

Why was ZeroVerse created?
  • ZeroID is leaking your data (IP, browser info, ...) into the normal internet
    •    See "ZeroID is not truly decentralised",
       and also the comments here and here,
       and ZeroSecurity.

  • privacy/anonymity:
    e.g. register your ID over BitMessage, and no one will ever know what kind of ZeroID you had before.
       Read more.

  • No need to restart ZeroNet anymore to change your ID.
       See the picture on top here.


How can I test if my ID works?

  • Check out all sites that support the ZeroVerse ID already (email, blogs, chats, ...).
